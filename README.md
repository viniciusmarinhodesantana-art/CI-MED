# CI-MED - Sistema de Agendamento Clínico

O **CI-MED** é um sistema em linguagem C desenvolvido para gerenciar o agendamento de consultas médicas em diferentes agências de atendimento regional.

## 📌 Funcionalidades Atualizadas
*   **Gerenciamento por Agências:** Suporte para unidades em Recife, Olinda, Paulista e Camaragibe.
*   **Controle de Turnos Dinâmico:** Listagem e validação de vagas (limite de 6 por turno) para os horários da manhã e da tarde.
*   **Cadastro de Pacientes:** Fluxo de coleta e armazenamento de dados do paciente (Nome e CPF) associados de forma segura via estruturas de dados aninhadas (`structs`).
*   **Interface via Terminal:** Menu interativo e modular estruturado com `do-while` e `switch-case`.

## 🛠️ Tecnologias e Conceitos Aplicados
*   **Linguagem C** (Consolidando lógica de programação e manipulação de arrays e strings).
*   **Estruturas de Dados Modulares:** Uso avançado de matrizes de caracteres e structs aninhadas (`Agencias` -> `Especialidade` -> `Turnos`).
